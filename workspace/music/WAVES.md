## Production Principles (The Rubin Rules)
1. **Serve the song.** Not the ego, not the trend, not the algorithm.
2. **Less is more.** If a track doesn't need something, don't add it.
3. **Trust the feeling.** If it hits, it hits. Don't overthink it.
4. **The album is a journey.** Track order matters as much as track selection.
5. **Stems are gold.** Any track with stems gets priority â€” we can reshape it.
6. **Duplicates are versions.** Multiple versions = the artist was searching. Find what they were looking for.
7. **Everything connects.** The Consciousness Series is one body of work across five albums.
